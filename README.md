🚗 RAAR Garage Web Application

A modern AI-powered garage management and customer service platform built using React, Vite, Express, and Google’s Generative AI.

The application is designed to help automotive service centers streamline customer interactions, vehicle service workflows, and digital garage operations through a fast, responsive, and user-friendly interface.

---

🌟 Features

🚘 Garage Service Management

* Vehicle service tracking
* Customer information management
* Service request handling
* Maintenance record organization

🤖 AI Integration

🚀 Installation

1. Clone the Repository

git clone https://github.com/RashneemaAmeen/Garage-web-app.git

2. Navigate to Project Directory

cd Garage-web-app

3. Install Dependencies

npm install

4. Configure Environment Variables

Create a .env file in the project root:

GOOGLE_API_KEY=your_google_api_key

---

▶️ Running the Application

Development Mode

npm run dev

The application will start at:

http://localhost:3000

---

📦 Production Build

Create an optimized production build:

npm run build

Preview the build:

npm run preview

---

🔍 Type Checking

Run TypeScript validation:

npm run lint
